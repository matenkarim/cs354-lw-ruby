    • Ruby is “…an interpreted, open-source programming language with a focus on simplicity and productivity.” Ruby is a popular programming language that enjoys widespread use by developers around the world (Stabile, 2023). Ruby is free of charge, free to use, copy, modify, and distribute. Ruby makes up the basis for the popular Ruby on Rails web application framework used in popular websites such as Twitter, GitHub, and Shopify.

    Variable Scope Identifiers

    Ruby uses symbols to identify the current scope and capitalization to identify mutability. Lowercase variables are mutable while variables starting with an uppercase letter are immutable. Here’s a quick run through:

  • myVar -> Local variable
  • MyVar -> Local constant
  • @myVar -> Instance variable
  • $myVar -> Global variable
  • self.myVar -> Refers to the current object’s version of the given variable or method (similar to Java’s “this” keyword)

    • Inheritance

    class MyCounter
     include Enumerable
    end

    Like Java, Ruby supports single inheritance where each child "class" can only have one parent. Ruby has specific terminology for determining which is which in the pecking order. Modules are the Ruby equivalent of parent “classes” which offer a collection of methods. Classes in Ruby are the children of their respective parent Module, extending its functionality.

    Object Structure

    2.times { print “Prepare for trouble!” }

    In Ruby, everything is an object! Every bit of information and code can be given their own properties and actions. Even numbers and primitive types are handled as objects in Ruby with their own methods and instance variables.

    Blocks

    def customBehaviorShowcase
     puts “Start of method…”
     yield if block_given? # Run any Blocks available
     puts “End of method…”
    End
    # Runs the method we just defined with a Block to demonstrate.
    customBehaviorShowcase do
     puts “Custom activity woohoo!”
    end
    # Output:
    # Start of method…
    # Custom activity woohoo!
    # End of method…

    Like Scheme, Ruby provides support for Closures (persistent scopes that hold onto local variables outside of their scope if a reference to their function is used later on) in the form of Blocks. You can attach a Block to any method to describe how the method should act (Members of the Ruby community, 2025).
