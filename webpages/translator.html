Available Translators/ Interpreters for Ruby

Ruby MRI (Matz's Ruby Interpreter) / CRuby

  • The default Ruby translator for versions up to Ruby 1.8
  • Developed by Ruby's creator, Yukihiro Matsumoto
  • Written in C
  • Community efforts to make frequent changes and updates, best for general programming

    • Installation

  • Ruby Download

    • YARV (Yet Another Ruby VM) / KRI (Koichi's Ruby Interpreter)

  • Developed by Koichi Sasada
  • Like Ruby MRI, it is written in C
  • Replaced Ruby MRI as the default translator for Ruby 1.9
  • Bytecode Interpreter attributed to the speed and efficiency of Ruby 1.9 compared to previous versions using AST

    • Installation

  • Ruby Download
  • Ruby Github

    • JRuby

  • Primarily written in Java and uses Java Virtual Machine
  • Supported compatibility with Ruby on Rails and Ruby MRI
  • Originally developed as a direct C port, but later added features to improve compile time
  • Being built on JVM, it has access to features like garbage collection and libraries

    • Installation

  • JRuby Download

    • mruby

  • Like Ruby MRI, mruby is created by Yukihiro Matsumoto
  • Primarily written in C, with some parts written in Ruby
  • Compatable with Ruby 2.1 and up to Ruby 3.4.3 (current stable version)
  • Can be linked and embedded in other programs
